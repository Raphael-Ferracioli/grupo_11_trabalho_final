# Estrutura do repositório

```text
goodfilms/
├── adrs/
├── dashboard/
├── data/
│   └── fixtures/
├── diagrams/
├── docs/
├── filmes/
├── goodfilms/
├── templates/
├── usuarios/
├── COMANDOS_GIT.md
├── README.md
├── manage.py
└── requirements.txt
```

## Pastas principais

- `adrs/`: decisões arquiteturais em Markdown.
- `diagrams/`: fontes Mermaid dos diagramas do documento.
- `docs/`: documentação técnica, incluindo OpenAPI.
- `filmes/`: domínio principal do sistema.
- `usuarios/`: perfil e autenticação.
- `dashboard/`: agregação de dados para tela inicial.
- `templates/`: interface HTML básica.
